# Security Policy

## Supported versions

Security fixes are not tracked by release branch yet. The latest code in `main` is the only version assumed to be maintained.

## Reporting a vulnerability

Please do not open a public issue for a security problem.

Report it privately here:

- Telegram: https://t.me/mansurov_rafael

Include:

- what the problem is
- how to reproduce it
- what devices or versions are affected
- screenshots, logs, or proof of concept if relevant

## What to expect

- we will review the report
- we may ask follow-up questions
- we will try to confirm and fix the issue before public disclosure
