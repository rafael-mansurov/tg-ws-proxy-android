# Локальный прокси для Telegram прямо на Android

- без регистрации
- без своего сервера
- без возни с конфигами
- включили тумблер, ткнули в Telegram, поехали

<br>
<br>

## Что это вообще такое

> [!NOTE]
> Это Android-приложение, которое поднимает локальный MTProto WebSocket proxy прямо на телефоне. То есть proxy крутится у вас на устройстве, а не где-то на чужом VPS в степях интернета.

Приложение само:

- запускает proxy на порту `1443`
- генерирует секрет
- отдает готовую ссылку для Telegram
- показывает QR-код
- умеет стартовать после перезагрузки телефона
- подсказывает, если Android душит его батарейной оптимизацией

<br>

## Плюсы

- работает прямо на телефоне
- не просит отдельный VPS
- не требует регистрации
- отдает готовую ссылку для Telegram
- запускается в пару тапов
- умеет жить в фоне

<br>

## Минусы

- Android может прибивать сервис в фоне
- на слабых или злых прошивках стабильность бывает так себе
- локальный proxy не всегда лучше хороших внешних proxy
- иногда без отключения оптимизации батареи все работает заметно хуже

## Установка

**01 · Скачиваете APK**

> Скачиваете [tg-ws-proxy.apk ≈24.8 МБ](https://github.com/rafael-mansurov/tg-ws-proxy-android/releases/download/latest-apk/tg-ws-proxy-release.apk).

**02 · Ставите на телефон**

> Ставите APK на Android `7.0+`. Если система ругается на установку из браузера, просто дайте разрешение на установку из неизвестных источников.

**03 · Открываете приложение**

> Открываете `TG WS Proxy` и видите главный экран с тумблером. Ничего руками настраивать не надо.

**04 · Включаете proxy**

> Тыкаете тумблер, и приложение поднимает локальный proxy на порту `1443`. Секрет оно тоже готовит само, без ручной возни.

**05 · Открываете Telegram**

> Нажимаете кнопку подключения. Дальше приложение само отдает ссылку `tg://proxy?...`, а Telegram подхватывает ее как обычный proxy.

**06 · Если Android начинает душить**

> Если приложение просит убрать оптимизацию батареи, лучше убрать сразу. Иначе сервис может просто отваливаться в фоне без предупреждений и выглядеть так, будто proxy сам по себе сломался.

**07 · Готово**

> Если все ок, Telegram подключается к proxy и дальше работает как обычно.

<br>

## Работает как обычно

Вы просто включаете proxy внутри приложения. Дальше Telegram получает обычную proxy-ссылку и подключается как умеет сам.  
Без ручной сборки ссылок. Без копипаста секрета. Без “сейчас я еще в заметки что-то сохраню”.

<br>

## Лайфхак

Если локальный proxy работает так себе, не надо героически сидеть на одном соединении и ждать, пока оно вдруг одумается.

Лучше так:

1. Идете во внешние proxy
2. Добавляете в Telegram как можно больше рабочих proxy
3. Оптимально держите штук `20`
4. После этого включаете в Telegram автоподключение proxy
5. Ставите таймаут `5` секунд

Тогда Telegram быстрее перебирает внешние варианты, если локальный proxy в этот момент тупит, лагает или просто не вывозит.

<br>

## Что важно после установки

- если приложение просит убрать оптимизацию батареи, лучше убрать
- если Android прибивает фоновые сервисы, proxy будет отваливаться
- если включите автозапуск, приложение сможет подниматься после перезагрузки телефона

<br>

## Как это работает внутри

Приложение поднимает локальный HTTP-сервер на порту `8080` и открывает интерфейс во встроенном WebView.  
Фоновый сервис слушает `0.0.0.0:1443`, а экран просто управляет этим сервисом через локальный API.

Короче: UI отдельно, proxy отдельно. Поэтому все не умирает от одного лишнего тапа по экрану.
