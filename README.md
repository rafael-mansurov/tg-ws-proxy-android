# TG WS Proxy

TG WS Proxy поднимает локальный MTProto WebSocket proxy прямо на Android-телефоне. Приложение не просит регистрацию, не гонит трафик через чужой сервер и за 1 тап отдает ссылку `tg://proxy` в Telegram.

## Что делает приложение

- Запускает proxy на телефоне на порту `1443`.
- Генерирует секрет на `32` hex-символа и хранит его в app storage.
- Отдает готовую ссылку для Telegram и показывает QR-код.
- Следит за скоростью, временем работы и последней успешной сессией.
- Умеет стартовать после перезагрузки телефона.
- Показывает предупреждение, если Android душит сервис оптимизацией батареи.
- Дает шаринг APK со ссылкой на релиз и картинкой `cover.jpg`.

## Как установить APK

Скачайте свежую сборку:

`https://github.com/rafael-mansurov/tg-ws-proxy-android/releases/download/latest-apk/tg-ws-proxy-debug.apk`

Дальше:

1. Установите APK на Android `7.0+`.
2. Откройте приложение.
3. Включите тумблер.
4. Нажмите кнопку подключения и откройте ссылку в Telegram.

## Как устроен запуск на Android

Приложение поднимает локальный HTTP-сервер на порту `8080` и открывает `ui/index.html` во встроенном WebView. Фоновый сервис слушает `0.0.0.0:1443`, а UI стучится в REST API и включает или выключает proxy без перезапуска экрана.

## Что проверить после запуска

1. Убедитесь, что статус в приложении стал `ON`.
2. Убедитесь, что Telegram открыл ссылку вида `tg://proxy?...`.
3. Убедитесь, что Android не режет приложение оптимизацией батареи.
4. Убедитесь, что после перезагрузки телефона автозапуск работает, если вы его включили.
