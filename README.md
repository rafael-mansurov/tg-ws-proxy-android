# Документация для tg-ws-proxy-android

## Описание проекта
 tg-ws-proxy-android — это приложение, которое позволяет использовать Telegram через прокси-серверы.

## Установка
1. Скопируйте или клонируйте репозиторий:
   ```bash
   git clone https://github.com/rafael-mansurov/tg-ws-proxy-android.git
   ```
2. Откройте проект в вашей IDE (например, Android Studio).
3. Следуйте инструкциям по сборке проекта.

## Использование
- Запустите приложение на вашем Android устройстве.
- Введите нужные параметры прокси-сервера.
- Используйте Telegram через ваше приложение.

## Поддержка
Если у вас возникли вопросы или проблемы, пожалуйста, создайте issue в репозитории или свяжитесь с автором.
