# Проект: tg-ws-proxy-android

## Обзор проекта
tg-ws-proxy-android является клиентом для работы с Telegram через WebSocket. Он обеспечивает доступ к API Telegram и позволяет вам взаимодействовать с приложениями и сервисами, использующими протокол WebSocket.

## Быстрый старт для Android
1. **Добавьте зависимость** в файл `build.gradle`:
   ```gradle
   implementation 'com.example:tg-ws-proxy-android:1.0'
   ```
2. **Инициализируйте клиент**:
   ```java
   TgWsProxyClient client = new TgWsProxyClient();
   client.connect();
   ```
3. **Используйте API**:
   ```java
   client.onMessage(message -> {
       // Обработка сообщения
   });
   ```

## Быстрый старт для VPS
1. **Установите необходимые зависимости**:
   ```bash
   sudo apt-get install ...
   ```
2. **Запустите сервер**:
   ```bash
   java -jar tg-ws-proxy-server.jar
   ```

## Архитектура
Проект состоит из двух основных компонентов: клиент Android и сервер VPS. Клиент отправляет запросы на сервер, который обрабатывает их и взаимодействует с API Telegram.

## API Endpoints
- `ws://<server_address>:<port>/ws` - WebSocket endpoint для подключения клиента.
- `http://<server_address>:<port>/api` - REST API для взаимодействия с сервером.

## Устранение неполадок
- **Ошибка подключения**: Проверьте, правильный ли адрес сервера и порт.
- **Прерывание соединения**: Убедитесь, что сервер работает, и ваш клиент правильно настроен.

## Замечания по безопасности
- Никогда не храните токены и пароли в открытом доступе.
- Используйте SSL для подключения к вашему серверу.
